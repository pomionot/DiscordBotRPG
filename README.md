    <ul>
        <li><code>!sta </code> キャラクターのステータスを表示するコマンド</li>

        <li><code>!fi</code>敵との戦闘をするコマンド</li>
        <li><code>!ga</code>武器が排出されるガチャガチャを引けるコマンド(武器の機能は未実装)</li>
